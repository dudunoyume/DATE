#使用說明
輸入檔案 點選需要轉換的程式 預設為 Excel 檔，可選擇csv檔

日期欄位名 : 輸入檔案內日期的欄位名 預設為 eventDate

年份格式：　想要輸出的年份格式

點擊開始輸出: 輸出成功 會出現完成的小視窗，如果出現問題不會有任何反應

日期格式可針對 年月日、無分隔符號、「,」、「/」、「\」、「-」、「.」 進行轉換
年分可轉換民國年、西元年

注意: 若原始檔案的格式，為無分隔符號，日月有僅為單位數的，如:2018112、107102，可能會出現，日期後綴為 error 的
